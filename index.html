Get access to the dom

  • document.getElementsByTagName()returns a collection of all of the elements within the DOM with that tag name
  • document.getElementsByClassName()returns a collection of all elements with a certain class
  • document.getElementById()returns the element with a given id
  • document.querySelector()returns the first element in the DOM that matches the given selector(s)
  • document.querySelector()
  • document.querySelector('p') returns the first <p>element
  • document.querySelector('.text-white')returns the first element with class text-white
  • document.querySelector('#art-001')returns the element with id attribute art-001
  • document.queryselector('p.text-white')returns the first <p> with class text-white
